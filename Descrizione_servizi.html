Descrizione dei servizi proposti

Di seguito cliccando i pulsanti verranno fornite le informazioni.

LOGIN - Entra

Dalla pagina http://localhost/phpcrud/loginp.php cliccando il pulsante "Entra" viene invocato in AJAX una chiamata alla pagina
http://localhost/phpcrud/servizi_bgnotes.php
Alla pagina sopra citata vengono passati in POST 2 parametri :
nome_servizio : "login" | semplice stringa
parametri : JSON.stringify(obj) | array formattato in una stringa di tipo JSON, dentro l'array inseriremo la nostra "email" e la nostra "password".

Il servizio risponderà con le variabili :
statusCode | codice di risposta della chiamata ( 404 risorsa non trovata, 200 richiesta andata con successo, 403 non autorizzato ecc )

success | data - conterrà la risposta che il servizio chiamato ci manderà se la richiesta è andata a buon fine

| data conterrà nel nostro caso "login ok" se l'utente è registrato ed email e password sono corretti, se l'esito è "login ok" verrà avviata un'altra chiamata in ajax chiamando
| "http://localhost/phpcrud/local_session.php", che inizializzerà la sessione.

error : data - conterrà il messaggio che ha causato l'errore nell'esito della richiesta.

esempio chiamata :
$.ajax({
type: "POST", url: "http://localhost/phpcrud/servizi_bgnotes.php",
data: { nome_servizio: "login",parametri:JSON.stringify(obj) },
success: function(data) {
// on successfull return it will alert the data
}

LOGIN - Log In With Google (Servizio di oAuth ESTERNO fornito da GOOGLE tramite apikey)

Dalla pagina http://localhost/phpcrud/loginp.php cliccando il pulsante "Sing In With Google" viene invocato in PHP la funzione $gClient->createAuthUrl();
$gClient è un API messa a disposizione da GOOGLE per l'autenticazione tramite oAuth
Per effettuare la chiamata è necessario creare un file di configurazione impostando le variabili richieste dal Google_Client();
setClientId : client id fornito in fase di registrazione al servizio oAuth sul sito di google dedicato ai developers.
setClientSecret : chiave segreta fornita in fase di registrazione al servizio oAuth sul sito di google dedicato ai developers.
setApplicationName : nome dell'applicazione per la quale è richiesta l'autenticazione tramite google.
setRedirectUri : URI di reindirizzamento nel caso in cui l'esito del login tramite google sia andato a buon fine. addScope : scope del servizio di oAuth

Reset Password ( servizio SMTP esterno fornito da GOOGLE tramite abilitazione dell'account di posta)

Dalla pagina http://localhost/phpcrud/reimposta_password.html cliccando il pulsante "Richiedi password" viene invocato in AJAX una chiamata alla pagina
http://localhost/phpcrud/servizi_bgnotes.php
Alla pagina sopra citata vengono passati in POST 2 parametri :
nome_servizio : "reset_password" | semplice stringa
parametri : JSON.stringify(obj) | array formattato in una stringa di tipo JSON, dentro l'array inseriremo la nostra "email" .

Il servizio risponderà con le variabili :
statusCode | codice di risposta della chiamata ( 404 risorsa non trovata, 200 richiesta andata con successo, 403 non autorizzato ecc )

success | data - conterrà la risposta che il servizio chiamato ci manderà se la richiesta è andata a buon fine

| data se la richiesta è andata a buon fine il messaggio di risposta del server sarà :"Se la mail è registrata riceverà una mail con la password.", a questo punto se la mail inserita è registrata verrà inviata una mail contenente la password.

error : data - conterrà il messaggio che ha causato l'errore nell'esito della richiesta.

Per l'invio della posta è stato configurato un server SMTP in locale impostando i parametri :
smtp_server= indirizzo del server SMTP di GOOGLE
smtp_port=porta per l'accesso al server SMTP esterno
auth_username= indirizzo di posta elettronica che si utilizzerà per inviare le mail - NECESSARIO PER L'AUTENTICAZIONE
auth_password= password dell'indirizzo di posta elettronica che si utilizzerà per inviare le mail - NECESSARIO PER L'AUTENTICAZIONE

esempio chiamata :
$.ajax({
type: "POST", url: "http://localhost/phpcrud/servizi_bgnotes.php",
data: { nome_servizio: "reset_password",parametri:JSON.stringify(obj) },
success: function(data) {
// on successfull return it will alert the data
}
});

Crea richiesta gruppo

Dalla pagina http://localhost/phpcrud/mappa.html cliccando il pulsante "Invia Richiesta" viene invocato in AJAX una chiamata alla pagina
http://localhost/phpcrud/servizi_bgnotes.php
Alla pagina sopra citata vengono passati in POST 2 parametri :
nome_servizio : "add_ricerca_gruppo" | semplice stringa
parametri : JSON.stringify(obj) | JSON.stringify(obj) | array formattato in una stringa di tipo JSON, dentro l'array inseriremo "nome","email","categoria","commento","latitudine","longitudine" .

Il servizio risponderà con le variabili :
statusCode | codice di risposta della chiamata ( 404 risorsa non trovata, 200 richiesta andata con successo, 403 non autorizzato ecc )

success | data - conterrà la risposta che il servizio chiamato ci manderà se la richiesta è andata a buon fine

| data se la richiesta è andata a buon fine il server risponderà con "Richiesta inserita con successo" e verranno salvati sul database i dati relativi alla richiesta.

error : data - conterrà il messaggio che ha causato l'errore nell'esito della richiesta.

esempio chiamata :
var obj = {};
obj.nome= $("#from_name").val();
obj.email = $("#from_email").val();
obj.telefono = $("#from_phone").val();
obj.categoria = $("#Materieinteressate").val();
obj.commento = $("#comments").val();
obj.latitudine=position.coords.latitude;
obj.longitudine=position.coords.longitude;
$.ajax({
type: "POST",
url: "http://localhost/phpcrud/servizi_bgnotes.php",
data: { nome_servizio: "add_ricerca_gruppo",parametri:JSON.stringify(obj) },
success: function(data) {
}
});
